# d2c-churn-model
# D2C Customer Churn Prediction

## Overview

This repository contains a Random Forest based churn prediction model for a D2C personal-care brand.

## Files

* churn_model.ipynb : Training and evaluation notebook
* model.pkl : Saved Random Forest model
* metrics.json : Model evaluation metrics
* requirements.txt : Required Python libraries

## Model Performance

* Accuracy: 0.7887
* F1 Score: 0.7788
* ROC AUC: 0.8617

## Usage

Open `churn_model.ipynb` and run all cells to reproduce the results.
